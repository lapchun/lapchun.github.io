  最近因为毕业论文和托福考试的压力，搞得有点心烦气躁，所以决定接触新事物散散心。同时为了之后博士期间有能力兼职做网页开发赚点外快，我现在就开始学习各大主流前端框架，从Vue.js开始。本文会逐步整理本人从入门到进阶学习所遇到的坑，以及相应的填坑方法。

Vue.js安装，最后“cnpm run dev”报错

我是根据菜鸟教程来进行Vue.js的安装，前面都很顺利，最后一步“cnpm run dev”报错，报错信息如下：

$ cnpm run dev  
> my-project@1.0.0 dev D:\vue\my-project 
> webpack-dev-server --inline --progress --config build/webpack.dev.conf.js

npm ERR! code ELIFECYCLE
npm ERR! errno 1
npm ERR! my-project@1.0.0 dev: `webpack-dev-server --inline --progress --config build/webpack.dev.conf.js`
npm ERR! Exit status 1
npm ERR!
npm ERR! Failed at the my-project@1.0.0 dev script.
npm ERR! This is probably not a problem with npm. There is likely additional logging output above.

npm ERR! A complete log of this run can be found in:
npm ERR!     C:\Users\LapChun\AppData\Roaming\npm-cache\_logs\2019-04-24T12_54_21_588Z-debug.log

解决方法：这是新版webpack存在的bug,先卸载现有的新版本webpack，再安装老版本就可以了。

cnpm uninstall webpack-dev-server
cnpm install webpack-dev-server@2.9.1
cnpm run dev

依次执行上述3条命令，问题得以解决。
